AwaitableCoroutine

AwaitableCoroutine

on 2021-05-08

AwaitableCoroutine は、async/await 構文を使用可能にしたコルーチンを提供する C# 向けライブラリです。 内部的にはTask-Like、Awaitable パターン、AsyncMethodBuilder が使われています。

  • AwaitableCoroutine - GitHub
  • AwaitableCoroutine - NuGet Gallery
  • AwaitableCoroutine.FSharp - NuGet Gallery
  • AwaitableCoroutine.Altseed2 - NuGet Gallery

Example

using System;

using AwaitableCoroutine;

var runner = new CoroutineRunner();

int count = 0;

var coroutine = runner.Create(async () => {
    Console.WriteLine("Started!");

    for (var i = 0; i < 10; i++)
    {
        count++;
        await AwaitableCoroutine.Yield();
    }
}).OnCompleted(() => Console.WriteLine("Finished!"));

while (true)
{
    runner.Update();
    if (coroutine.IsCompleted) break;

    Console.WriteLine($"{count}");
}
